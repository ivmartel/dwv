WindowLevel class.
Pseudo-code:

 if (x <= c - 0.5 - (w-1)/2), then y = ymin
 else if (x > c - 0.5 + (w-1)/2), then y = ymax,
 else y = ((x - (c - 0.5)) / (w-1) + 0.5) * (ymax - ymin) + ymin
